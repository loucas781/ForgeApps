# ForgeApps

Shared support and legal pages for the Forge app suite.

## GitHub Pages

Enable Pages in this repo:

- Settings -> Pages
- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

Then use these URLs in App Store Connect:

- Support URL: `https://loucas781.github.io/ForgeApps/`
- Privacy Policy URL: `https://louisbirch.co.uk/privacypolicy/`
- Terms of Use URL: `https://loucas781.github.io/ForgeApps/terms.html`
